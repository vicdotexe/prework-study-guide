Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Title is required and useful for search engines.
  • Keep header elements in order.
  • Elements have a parent/child relationship.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "message": commits any changes with a message
  • git pull origin branch-name: pulls files
  • git push origin branch-name: pushes changes
